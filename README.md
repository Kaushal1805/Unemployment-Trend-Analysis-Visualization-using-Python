📊 Unemployment Trend Analysis (India)

This project analyzes unemployment trends across 20+ Indian states using Python and data visualization techniques. The goal is to uncover patterns, compare regional impacts, and understand the effect of COVID-19 on employment.

🔍 Key Objectives
Analyze state-wise unemployment data
Compare pre- and post-COVID trends
Identify high-risk regions with rising unemployment
Derive actionable insights from data
🛠️ Tools & Technologies
Python (Pandas, NumPy)
Matplotlib & Seaborn
Jupyter Notebook
📈 Project Highlights
Cleaned and processed raw unemployment datasets for accurate analysis
Visualized trends showing a 3x spike in unemployment during 2020 (COVID period)
Identified top 5 states with highest unemployment using comparative analysis
Discovered that urban areas recovered ~2x faster than rural areas post-lockdown
💡 Key Insights
COVID-19 had a significant nationwide impact on employment
Recovery rates varied strongly by region and urbanization
Certain states consistently showed higher unemployment trends
🚀 Outcome

This project demonstrates practical skills in data cleaning, exploratory data analysis (EDA), and visualization, along with the ability to convert raw data into meaningful business insights.
